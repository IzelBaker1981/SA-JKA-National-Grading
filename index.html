Important reminders

  • Development students only see development fee values, and standard students only see standard fee values.
  • The Japanese grading form must be printed before payment and final submission are unlocked.
  • The reference number uses the selected dojo registration number plus the sequence starting from 010.
  • Submissions can be exported to Excel/CSV for later download from GitHub Pages.

Submission register

GitHub Pages cannot directly write to a server-side Excel file or automatically email/WhatsApp attachments on its own. This version generates the populated print pack, a mail draft, a WhatsApp draft, and a downloadable submissions register CSV so the workflow stays fast and reliable in a static GitHub environment.
