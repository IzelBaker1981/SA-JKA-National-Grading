Repository-friendly workflow

This app is fully static, so it can run on GitHub Pages without a server. Keep the file as index.html in the repository root, and GitHub Pages will publish it from the selected branch or /docs folder.

Use the Print / Save PDF button to keep a basic paper trail of the final application view.
